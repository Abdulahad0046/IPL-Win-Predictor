# 🏏 IPL Win Predictor

AI-Powered Live Match Prediction System built using Machine Learning and Flask.

## Features

- Live IPL Match Winner Prediction
- Random Forest Classifier
- Win Probability Calculation
- Match Situation Analysis
- Run Rate Analysis
- Modern Interactive Dashboard
- Flask Web Application

## Tech Stack

### Frontend
- HTML
- CSS
- Jinja2

### Backend
- Flask

### Machine Learning
- Python
- Pandas
- NumPy
- Scikit-Learn
- Random Forest Classifier

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. Model Training
5. Flask Integration
6. Web Deployment
